# Security Policy

## Supported Scope

This repository is an infrastructure starter. Security reports are especially relevant for:
- exposed secrets
- unsafe default configuration
- TLS or proxy misconfiguration
- data leakage through tracked runtime files
- insecure Docker or network defaults

## Reporting

If you find a security issue, please report it privately first.

Contact:
- Website: <https://www.adrielzimbril.com/>
- GitHub: <https://github.com/adrielzimbril>

Please include:
- a short summary
- impact
- reproduction steps if applicable
- affected files or configuration areas

## Disclosure

Please avoid opening a public issue for sensitive vulnerabilities until there is time to assess and remediate the problem.
