# Contributing

Thanks for contributing.

## Scope

This repository is focused on infrastructure, Docker configuration, runtime orchestration, and deployment-friendly templates.

Good contributions include:
- documentation improvements
- Docker and Compose cleanup
- safer defaults
- portability improvements
- security hardening
- clearer environment variable documentation

## Before opening a PR

Please check:
- no real secrets are committed
- `.env` is not tracked
- runtime certificates, ACME data, and logs are not included
- changes stay generic enough for a reusable starter

## Pull Request Guidelines

- keep changes focused
- explain the reason for the change
- update docs when behavior changes
- avoid bundling unrelated refactors
